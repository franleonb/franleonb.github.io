# franleonb.github.io
¿Alguna vez te has preguntado cómo sería la canasta de Caperucita Roja si fuera vegana? Este reportaje explora la canasta básica vegana frente a la tradicional, mostrando cómo cambian los alimentos, qué productos se sustituyen o eliminan, y qué desafíos enfrentan quienes siguen esta dieta.
